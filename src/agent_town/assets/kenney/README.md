# Kenney Asset Slice

Selected spritesheets extracted from the local Kenney zip files in the project root.

- `characters.png` from `kenney_roguelike-characters.zip`
- `rpg_tiles.png` from `kenney_roguelike-rpg-pack.zip`
- `emotes.png` and `emotes.xml` from `kenney_emotes-pack.zip`

The source zip license files identify these assets as Creative Commons Zero (CC0). Credit to Kenney or www.kenney.nl is appreciated but not required.
