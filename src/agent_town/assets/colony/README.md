# Colony sprite set

Authored sprites for the colony viewer (`colony_view.py`), 25px base tiles.

## Provenance / license

- **pawn_00.png .. pawn_23.png** — the 24-character "Tiny Characters Set",
  sliced from `32_Characters/All.png` (6x4 grid, trimmed to content).
  **CC0**, Fleurman (based on GrafxKid's work) —
  https://opengameart.org/content/tiny-characters-set . No attribution required;
  crediting Fleurman + GrafxKid is best practice.
- **house*.png, tree*.png, tile_*.png, wall.png, gate.png, crate.png, barel.png,
  tileset.png, scene.png** — provided by Kayden, matching the ansimuz
  `town_rpg_pack` / `trees_and_bushes_pack` style (both **CC0**, Luis Zuno).

## Source packs in `assets/` (zips)

CC0-confirmed:
- `town_rpg_pack.zip`, `trees_and_bushes_pack.zip` — Luis Zuno / ansimuz.
- `32_Characters.zip` — Tiny Characters Set, Fleurman / GrafxKid (used for pawns).
- `FreePixelFood.zip` — 64 food icons, Henry Software (assets CC0, code MIT),
  https://henrysoftware.itch.io/pixel-food . Cleared for HUD/good icons.
- `kenney_*` packs — Kenney, always CC0.

License unverified — do **not** ship: `gfx.zip` (`Overworld.png`/`character.png`
look like rips; treat as suspect).
