# Local Agent Town - Visual Design

**Last reviewed:** 2026-06-26  
**Status:** active

This project follows the `LLM_Workbench` dark-mode visual baseline, adapted for a desktop simulation.

## Palette

| Name | Hex | Role |
|---|---:|---|
| Onyx | `#0A0A0A` | App background |
| Carbon Black | `#191919` | Panel surfaces |
| Iron Grey | `#474747` | Borders and muted strokes |
| Bright Snow | `#F7F7F7` | Primary text |
| Cerulean | `#3A7CA5` | Selected state and primary accent |
| Medium Jungle | `#4DAA57` | Social and healthy state accents |
| Old Gold | `#E0BD3E` | Food and warning accents |
| Rose Punch | `#CF4F84` | Personal or social accents |
| Vibrant Turquoise | `#1ABCBD` | Live state accents |

## Viewer Rules

- Keep the main experience as the first screen.
- Use the map as the primary visual surface.
- Use the bundled Kenney sprites for agents, place markers, and short-lived emotes when available.
- Keep the inspection panel compact and scannable.
- Do not add a landing page, web UI, or marketing screen.
- Do not use emoji as icons.
- Avoid decorative effects that reduce map readability.
